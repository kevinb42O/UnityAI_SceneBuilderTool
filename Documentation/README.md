# Movement Controller Package

Welcome to the **Movement Controller** package! This AAA-quality movement system is designed to provide developers with a robust and flexible framework for character movement in Unity. Whether you're creating a fast-paced shooter, a platformer, or a realistic simulation, this package has you covered.

## Overview

The Movement Controller package includes:

- **AAA Movement System**: A highly configurable movement system that allows for smooth and responsive character control.
- **Advanced Crouch/Slide System**: Features for crouching and sliding, enhancing gameplay mechanics and player experience.
- **Comprehensive Documentation**: Detailed guides and references to help you get started and make the most of the package.

## Quick Start

To get started with the Movement Controller package, follow these simple steps:

1. **Import the Package**: Download and import the Movement Controller package into your Unity project.
2. **Add the Movement Controller**: Drag the `MovementController` prefab into your scene.
3. **Configure Settings**: Open the `MovementConfig` and `CrouchConfig` scripts to adjust settings according to your game's needs.
4. **Test Your Setup**: Play the scene to see the movement controller in action!

## 5-Minute Setup Guide

1. **Import the Package**: 
   - Download the package from the Unity Asset Store.
   - In Unity, go to `Assets > Import Package > Custom Package` and select the downloaded file.

2. **Add the Movement Controller**:
   - Navigate to the `Assets/Scenes` folder and open `Example_Scene.unity`.
   - Drag the `MovementController` prefab from the `Assets/Scripts/MovementController` folder into your scene.

3. **Configure Movement Settings**:
   - Open `Assets/Scripts/MovementController/MovementConfig.cs` and adjust the parameters to fit your gameplay style.
   - Open `Assets/Scripts/MovementController/CrouchConfig.cs` to set up crouching and sliding behaviors.

4. **Play and Test**:
   - Click the Play button in Unity to test the movement and crouch functionalities.
   - Adjust settings as needed based on your testing.
